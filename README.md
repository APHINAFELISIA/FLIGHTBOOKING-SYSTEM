# FLIGHTBOOKING-SYSTEM
The Flight Booking System is a web-based application designed to manage airline reservations efficiently. The system allows users to search for available flights, view flight details, and book tickets. It simulates the process of booking airline tickets and managing passenger information using database concepts.
In modern transportation systems, airline booking systems play a crucial role in managing large amounts of data such as passenger details, flight schedules, and bookings. This project demonstrates how Database Management Systems (DBMS) can be used to store, retrieve, and manage this data efficiently.

The project is developed using HTML, CSS, JavaScript, and database concepts, providing a simple interface for users to interact with the system.
# PROBLEM STATEMENT
The Flight Booking System is designed to solve this problem by providing a web-based platform that manages flight information and booking processes efficiently using database management concepts.
# OBJECTIVES
The main objectives of this project are:
To develop a web-based flight booking system.
To apply Database Management System concepts in a real-world application.
To allow users to search and view available flights.
To maintain records of users, flights, and bookings.
To create a simple and user-friendly interface for airline booking.
To demonstrate data storage, retrieval, and management using a database.

# FEATURES
# 1. Flight Search

The system allows users to search for available flights by entering the departure city, destination city, and travel date. The system retrieves matching flights from the database.

# 2. Flight Information Display

Users can view detailed flight information such as:

Airline name

Departure time

Arrival time

Flight duration

Ticket price

# 3. Passenger Management

The system stores and manages passenger details including:

Passenger name

Email address

Phone number

Passport number

# 4. Booking Management

Users can book flights and the system records the booking details including:

Booking ID

Passenger ID

Flight ID

Booking date

Booking status

# 5. Ticket Generation

After booking a flight, the system generates a ticket containing:

Seat number

Travel class

Ticket price

# 6. Payment Processing

The system records payment details for each booking including:

Payment method

Amount paid

Payment date

Payment status

# 7. Flight Status Tracking

The system stores and displays the status of flights such as:

Scheduled

Delayed

Cancelled

# 8. User Authentication

The system includes a login interface that allows users to access their booking information securely.

# 9. Database Management

The system efficiently manages data using relational database tables such as:

Airports

Airlines

Flights

Passengers

Bookings

Tickets

Payments

# 10. User-Friendly Interface

The application provides a simple and responsive user interface that allows users to easily search, view, and book flights.
